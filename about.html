🚀 Космоматематика

Бесплатная обучающая игра по математике для дошкольников и первоклассников. Некоммерческий семейный проект.

Конфиденциальность

Приложение не собирает, не передаёт и не хранит никакую персональную информацию — ни детей, ни взрослых.

Весь игровой прогресс, настройки и достижения хранятся только локально на вашем устройстве (localStorage браузера). Аналитика, реклама, трекеры, cookies и сторонние сервисы сбора данных не используются. Аккаунтов, чатов, платежей и определения местоположения в приложении нет.

После установки на домашний экран приложение работает полностью офлайн.

Если вы очистите данные браузера или удалите приложение, прогресс будет безвозвратно удалён. Чтобы перенести прогресс на другое устройство, используйте «Код спасения» в разделе «Профиль».

Приложение размещено на GitHub Pages. GitHub как хостинг-провайдер может обрабатывать стандартные серверные логи (IP-адреса запросов) согласно собственной политике конфиденциальности. Автор приложения доступа к этим логам не имеет.

Условия использования

Приложение предоставляется бесплатно для личного некоммерческого использования, «как есть» (as is), без каких-либо гарантий. Автор не несёт ответственности за потерю данных, проблемы с устройством или перерывы в работе.

Весь код, иллюстрации и контент принадлежат автору, если не указано иное. Использование материалов без разрешения не допускается.
